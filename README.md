# Stickies

A minimal, beautiful floating sticky-notes app for macOS. Built in public.

- **Design spec:** [`docs/2026-04-22-stickies-design.md`](docs/2026-04-22-stickies-design.md)
- **Implementation plan:** [`docs/2026-04-22-stickies-plan.md`](docs/2026-04-22-stickies-plan.md)

v1 not yet shipped — plan is being executed task-by-task. Once scaffolded, build with:

```
brew install xcodegen
xcodegen generate
open Stickies.xcodeproj
```
